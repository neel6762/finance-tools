# Finance Tools

Personal finance calculators and simulators built with Next.js.

## Tools

- **Investment Simulator** — Compound growth projections with contributions, life events, Monte Carlo analysis, and nominal/real value toggle (CAD/INR)
- **Mortgage Simulator** — Canadian mortgage calculator supporting 6 payment frequencies with amortization schedules, equity buildup charts, and principal vs interest breakdown
- **Retirement Simulator** — Model retirement withdrawals, fund longevity, pension income, and life events with inflation-adjusted projections
- **Goal Planner** — Calculate monthly SIP needed to reach a financial goal by a target age, with step-up contributions and scenario comparison

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech Stack

- Next.js 14 (App Router)
- React 18
- Tailwind CSS
- Recharts 3
- TypeScript
